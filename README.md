# 🪙 Conversor de Monedas - Challenge Alura ONE

## 📝 Descripción
Este proyecto es un conversor de monedas de consola desarrollado en Java. Es el resultado del primer Challenge de la ruta de Backend en el programa **Oracle Next Education (ONE)** en alianza con **Alura Latam**. La aplicación permite a los usuarios realizar conversiones en tiempo real entre diferentes divisas de América Latina y el mundo, consumiendo una API externa.

## 🚀 Características
- **Conversión en tiempo real:** Utiliza datos actualizados al momento de la consulta.
- **Menú interactivo:** Interfaz de consola amigable e intuitiva.
- **Múltiples divisas:** Soporta la conversión entre el Dólar (USD), Peso Argentino (ARS), Real Brasileño (BRL) y Peso Colombiano (COP).
- **Historial de transacciones:** Registra cada conversión realizada durante la sesión, detallando la cantidad, la moneda de origen, la moneda de destino y la marca de tiempo exacta (fecha y hora).

## 🛠️ Tecnologías y Herramientas Utilizadas
- **Java** (Lenguaje principal)
- **Programación Orientada a Objetos (POO)**
- **Clase HttpClient** (Para peticiones web nativas de Java)
- **Gson (Google)** (Librería para el parseo y manipulación de archivos JSON)
- **ExchangeRate-API** (API externa para obtener las tasas de cambio)
- **Clase Record** (Estructura inmutable para el manejo de los datos de la API)

## ⚙️ Cómo ejecutar el proyecto
1. Clona este repositorio en tu máquina local.
2. Abre el proyecto en tu IDE preferido (VS Code, IntelliJ, Eclipse).
3. Asegúrate de tener agregada la librería `gson-2.10.1.jar` en las dependencias de tu proyecto.
4. Ubica y ejecuta el método `main` dentro de la clase `ConversorApp.java`.
